# Basics about MongoDB on Linux

# Set up your database in a few minutes!

Well, for most of the time, when we had installed MongoDB on the server, we will find the program folder is in /usr/local/mongodb/* and the binary files in /usr/local/mongodb/bin/*. If we cd into the binary executable file folder, we will see some important programs. The first is mongod, which is used for starting a database server. And the second one is mongo, which is used to establish a connection between user and database. It provides a command-line user interface like MySQL

To start the database server, we should first make a folder to store our database file. sudo mkdir -p data/db It will create a data folder at the root level of the system, and there is a db folder inside to store database files.

It is quite easy to start! Just go to /usr/local/mongodb/bin/ and type ./mongod You will see a program runs on your command line!

But that is not what we want so far! We would like it to run as a service in Linux system. As a result, we need to add more configuration. First, we need to add configeration file for mongod launching. type vim mongodb.conf in bin folder.

We simply add the following code:

# dbpath=/usr/local/mongodb/data/db/  It is just a place to store data, normally it will store in /data/db/, anyway, it's up to you!
logpath=/var/log/mongodb.log     # Place to store log files, IMPORTANT!
logappend=true
port=27017 # By default
fork=true # IMPORTANT, To keep application run in back ground
auth=true # If you wish to add more account more than root

Super Good! Now let's add configuration to make it as a service when boot! First we create file sudo vim /usr/lib/systemd/system/mongodb.service

Then we add some code:

[Unit]
    Description=mongodb 
    After=network.target remote-fs.target nss-lookup.target
     
[Service]
    Type=forking
    ExecStart=/usr/local/mongodb/bin/mongod --config /usr/local/mongodb/bin/mongodb.conf
    ExecReload=/bin/kill -s HUP $MAINPID
    ExecStop=/usr/local/mongodb/bin/mongod --shutdown --config /usr/local/mongodb/bin/mongodb.conf
    PrivateTmp=true
      
[Install]
    WantedBy=multi-user.target

We are about to finish, then we can reboot the system service and start database services!

systemctl daemon-reload
systemctl start mongodb.service
