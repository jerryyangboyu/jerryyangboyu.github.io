# Quick Establish Your SSH Connection To Your Remote Server

SSH is an unsymmetric encryption technique widely used for building a connection between the client-side and remote server. You may also use it for push and fetch code from the SVN server.

If you don't what unsymmetric encryption is. Well, it consisted of the public and private keys. The public key is usually stored on the server, and the private key is stored on the client-side. It like everyone can send an email to a mailbox, but only the person who holds the private key can open the mailbox.

# Quick implementation: A few steps

# Set up the SSH server

Make sure your remote Linux server has a pre-installed SSH server, or you should google and install it by yourself. ( Mine has pre-installed in CentOS7, unname -r to check system version)

Well, you should add configuration to SSH config file vim /etc/ssh/sshd_config change PasswordAuthentication no to PasswordAuthentication yes and save.

Well, your remote-server provider will give you a .pem private key for ssh connection use. keep it safe and we will use it later.

WARNING This document is not complete for the general remote Linux server. The document can only be implemented Aliyun Node.js CentOS server

# Set up your ssh client for Windows platform

WARNING This section is for the Windows platform only. You should set up developer mode first and add ssh function to your operating system

Edit or add config textfile in C:/Users/<username>/.ssh/ folder. type following config:

Host linux  #Nick name for your remote server
User root #user you want to log in
Hostname 123.56.13.127 #IP address
IdentityFile <your ssh key address>

Now open terminal(Powershell). and type ssh linux You will see the following prompt:

> ssh linux
Last login: Tue Mar 31 07:54:13 2020 from <Remote IP address>

Welcome to Alibaba Cloud Elastic Compute Service !       

[root@<Your server> ~]#

It means you had log into the server successfully!
