We've got what you need!

Jelly Note is an extracurricular computer learning reference based on textbooks, past papers, and extracurricular materials. It can be used both for the learning of new computer knowledge and for reviewing existing knowledge. It will solve the problem for you on the way to your success, so that you can prepare for the CAIE standardized test more comprehensively. At present, Jelly-Note has achieved a comprehensive coverage of CAIE key knowledge through practice, and has been recognized by computer students. The project is currently being maintained by Jerry and Michael. Please remember our website www.yangboyu.net
